# OpenMetadata Google Agent

## Running the Agent from `src` Using Google ADK

To run the agent from the `src` directory using the Google ADK, use the following command:

```sh
uv run adk web
```

This repository is an exemplary implementation of an LLM agent that demonstrates the usage of the OpenMetadata MCP server.

### Prerequisites

Ensure the following before running the agent:
- All dependencies are installed.
- You are in the correct directory.
- Environment variables `OM_JWT_TOKEN` and `OM_URL` are set.

### Additional Notes

This module provides instructions to run the agent from the `src` directory using Google ADK.
